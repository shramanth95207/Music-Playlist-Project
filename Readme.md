Music Playlist Manager (Python Project)

This project is a simple Python-based music playlist manager that allows
users to create and manage their own playlist. Users can add songs,
delete songs, search for songs, play a selected song, and navigate using
next/previous controls. It is built without HTML, using only Python.

FEATURES - Add songs to playlist - Remove songs - Search songs by name -
Play selected song - Next / Previous song controls - Display complete
playlist - Screenshot support for project demonstration

TECHNOLOGIES USED - Python 3 

PROJECT STRUCTURE Music Playlist/ │ ├── main.py # Main program 
├── playlist_manager.py # Playlist operations ├── player.py #
Play, next, previous logic ├── README.md # Documentation 

HOW TO RUN Terminal: python main.py

VS Code: 
1. Open folder 
2. Open main.py 
3. Run



NOTES - Works fully offline - No HTML - Can use CLI or WxPython GUI
